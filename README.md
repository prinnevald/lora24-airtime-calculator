# SX128x LoRa 2.4 GHz Airtime Calculator

A static browser version of the Python airtime calculator.

## Files

- `index.html` — page structure
- `style.css` — styling
- `script.js` — airtime calculation logic and form handling
- `.nojekyll` — tells GitHub Pages to serve the static files directly

## Local preview

Open `index.html` in your browser.

## GitHub Pages deployment

1. Create a GitHub repository.
2. Upload these files to the repository root.
3. Go to **Settings → Pages**.
4. Under **Build and deployment**, set **Source** to **Deploy from a branch**.
5. Select the `main` branch and `/root`.
6. Save.
7. Open the published URL after GitHub finishes deployment.
